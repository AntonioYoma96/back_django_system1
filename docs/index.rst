SmartPeople 2 (BACKEND)
=======================
Versión: |release|

----

SmartPeople 2 (BACKEND) es la estructura de datos y API para la nueva versión de SmartPeople. En este recurso se
documenta todos los modelos y endpoints para el sistemas, asi como cualquier configuración adicional que se haya
aplicado. Además incorpora los requerimientos, configuración e instalación para el backend, y un listado con las notas
de actualización.

Para navegar por el documento, use al panel de navegación a la izquierda.
